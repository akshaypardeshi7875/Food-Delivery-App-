# 🍔 Food Delivery Web Application — React Project

A modern and responsive **Food Delivery Web Application** built using **React.js** and **Bootstrap 5**. This project focuses on building scalable UI components, smooth navigation, and a clean user experience similar to real-world food ordering platforms.

---

## 🚀 Features

* Responsive food delivery interface (Desktop + Mobile)
* Modern UI with reusable React components
* Dynamic food menu with card-based layout
* Category-based menu sections
* Smooth navigation using React Router
* Clean and scalable folder structure
* Optimized layout using Bootstrap Grid system

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript (ES6)
* **Styling:** Bootstrap 5, CSS3, HTML5
* **Routing:** React Router
* **Build Tool:** Vite
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/akshaypardeshi7875/Food-Delivery-App-.git
```

Navigate to project folder:

```bash
cd Food-Delivery-App-
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
